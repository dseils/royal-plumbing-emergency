# Royal Plumbing Emergency Landing Page

**Client:** Royal Plumbing Pros Parker  
**Purpose:** Emergency plumbing service landing page optimized for Google Ads  
**Target Keywords:** emergency plumber parker co, 24/7 plumber parker, burst pipe parker  

## Page Structure

- **Hero Section:** 3-part headline system (pattern interrupt → mechanism → outcome)
- **Problem Agitation:** Cost comparison ($200 vs $15K damage)
- **Why Us:** 3 key differentiators (local, stocked trucks, upfront pricing)
- **Services:** 6 emergency services clearly listed
- **Social Proof:** Google reviews + testimonial
- **Final CTA:** Mobile-optimized tap-to-call

## Features

- Mobile-first responsive design
- Tap-to-call buttons throughout
- Fast loading (TailwindCSS CDN)
- Animated call button
- Clear value proposition above fold

## Deployment

Deploy to Vercel for fast, free hosting:

```bash
vercel --prod
```

## Local Development

Open `index.html` in any browser - no build step required.

## Conversion Optimization

- Primary CTA: Phone call (not form)
- Above fold CTA visible within 2 seconds
- Trust signals throughout (licensed, insured, local)
- Urgency messaging (damage costs, response time)
